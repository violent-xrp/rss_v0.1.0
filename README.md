## Why Split Licenses

The Pact and the code are two different assets:

- **The code** is an engine. It is collaborative. GPL enables forks, pull requests, and derivative works. Commercial licensing enables contractor use.
- **The Pact** is a constitution. It is sovereign. CC BY-ND ensures the canonical text belongs to T-0 and cannot be forked into competing versions.

The engine is open. The constitution is singular. That is by design.
